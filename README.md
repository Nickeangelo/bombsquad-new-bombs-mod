# BombSquad – New Bombs Mod
Este repositório contém o mod **New Bombs Plus** para BombSquad.

## Bombas incluídas
- **Gravity Bomb** – explosão gravitacional com skin azul
- **Pulse Nova** – explosão de pulso com skin neon
- **Nuclear Core** – explosão massiva com efeito radioativo

## Instalação automática dentro do jogo
1. Abra **Settings > Advanced > Scripts** no BombSquad Pro.
2. Cole o conteúdo de `installer.py`.
3. Rode o script.
4. O jogo instalará o mod automaticamente.
